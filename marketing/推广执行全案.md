✅ "新鲜鸡胸肉"——OK
❌ "肉类及其制品"——不OK，太模糊

你愿意吃的，才能给毛孩子吃。
这个标准最简单也最可靠。

#宠物零食怎么选 #配料表 #养狗必看 #避坑指南

## 六、数据追踪部署

### Google Analytics 4 部署
将以下代码添加到所有页面的 `</head>` 之前：

```html
<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```

### 关键转化事件埋点
```javascript
// 报名按钮点击
document.querySelectorAll('[data-track="register"]').forEach(btn => {
  btn.addEventListener('click', () => {
    gtag('event', 'click_register', { event_category: 'conversion' });
  });
});

// 产品购买点击（跳转小程序/微信）
document.querySelectorAll('[data-track="buy"]').forEach(btn => {
  btn.addEventListener('click', () => {
    gtag('event', 'click_buy', { event_category: 'conversion' });
  });
});

// 社群入口点击
document.querySelectorAll('[data-track="community"]').forEach(btn => {
  btn.addEventListener('click', () => {
    gtag('event', 'click_community', { event_category: 'engagement' });
  });
});
```

> ⚠️ 需要你做的：去 analytics.google.com 创建账号 → 获取测量ID → 替换 `G-XXXXXXXXXX`
